Quaternion Difference
"Difference" being defined as the angular displacement from one orientation to another: for orientations a and b, we can compute the angular displacement d which rotates from a to b.
$ad = b$
Quaternion Difference
Solving for $d$
$a^{-1}(ad) = a^{-1}b$
$(a^{-1}a)d = a^{-1}b$
$[1 \enspace 0]d = a^{-1}b$
$d = a^{-1}b$
Note: mathematicaly, it's closer to division than difference
Quaternion Dot Product
Notation and definition is similar to vector dot product
$ q_1 \cdot q_2 = \left[\begin{smallmatrix} w_1 && v_1 \end{smallmatrix}\right] \cdot \left[\begin{smallmatrix} w_2 && v_2 \end{smallmatrix}\right] = w_1w_2 + v_1 \cdot v_2 $
$ q_1 \cdot q_2 = \left[\begin{smallmatrix} w_1 && (x_1 && y_1 && z_1) \end{smallmatrix}\right] \cdot \left[\begin{smallmatrix} w_2 && (x_2 && y_2 && z_2) \end{smallmatrix}\right] $
$ = w_1w_2 + x_1x_2 + y_1y_2 + z_1z_2 $
Notes
  • Like the vector dot product, the result is a scalar.
  • For unit quats $a$ and $b$, $-1 \leq a \cdot b \leq 1$
  • Interested in absolute value of $a\cdot b$ because $a\cdot b = -(a\cdot -b)$
  • Geometric intepretation is similar to the interpretation of the vector dot product: the larger the absolute value, the more similar the angular displacements of $a$ and $b$
