Matrix Linear Transformations
Object and coordinates space, rotation
We can transform an object
We can transform the coordinates
Recall that linear transformations do not include translation.
Transforming an object by an given amount is equal to transforming the coordinate system by the opposite amount.
Multiple transformations are applied in the opposite order.
Example:
If we scale an object by 200% and then rotate it 20 degrees clockwise, this is equal to rotating the coordinate system 20 degrees counterclockwise and then scaling by 50%.
Rotation in 2D
When rotating in 2D we rotate around a point.
For now, consider rotating about the origin.
Only one param $\theta$, which is the amount of rotation.
Counterclockwise is normally positive while clockwise is negative.
Consider basis vectors $p = [1,0]$ and $q = [0, 1]$ rotating counterclockwise. The new values $p' = [cos \theta, sin \theta]$ and $q = [-sin \theta, cos \theta]$ So we have the matrix $R(\theta) = \left[ \begin{smallmatrix} p' \\ q' \end{smallmatrix} \right] = \left[ \begin{smallmatrix} cos \theta && sin \theta \\ -sin \theta && cos \theta \end{smallmatrix} \right] $
Rotation in 3D
Rotation in 3D is about an axis.
When we rotate by an amount $\theta$, we need to know which way is positive.
The standard way is the left-hand rule.
Rotation in 3D: left hand rule
The left hand rule works as such- put your hand in a thumbs up position.
Your thumb points towards the positive end of the axis of rotation.
Positive rotation about said axis is in the direction that your fingers curl.
Rotation in 3D axes
$p = [1,0,0]$
$q = [0,1,0]$
$r = [0,0,1]$
Rotation in 3D about the x-axis
$R_x(\theta) = \left[ \begin{smallmatrix} p' \\ q' \\ r' \end{smallmatrix} \right] = \left[ \begin{smallmatrix} 1 && 0 && 0 \\ 0 && cos \theta && sin \theta \\ 0 && -sin \theta && cos \theta \end{smallmatrix} \right] $
Rotation in 3D about the y-axis
$R_y(\theta) = \left[ \begin{smallmatrix} p' \\ q' \\ r' \end{smallmatrix} \right] = \left[ \begin{smallmatrix} cos \theta && 0 && -sin \theta \\ 0 && 1 && 0 \\ sin \theta && 0 && cos \theta \end{smallmatrix} \right] $
Rotation in 3D about the z-axis
$R_z(\theta) = \left[ \begin{smallmatrix} cos \theta && sin \theta && 0 \\ -sin \theta && cos \theta && 0 \\ 0 && 0 && 1 \end{smallmatrix} \right] = \left[ \begin{smallmatrix} p' \\ q' \\ r' \end{smallmatrix} \right] $
