Reflection
Also known as mirroring
Flips an object about a line (2D) or plane (3D)
Accomplished by scaling by -1
Reflection (2D)
$p_n = S(n, -1) = \left[ \begin{smallmatrix} 1+(-1-1)n_x^2 && (-1-1)n_xn_y \\ (-1-1)n_xn_y && 1+(-1-1)n_y^2 \end{smallmatrix} \right] $
$= \left[ \begin{smallmatrix} 1-2n_x^2 && -2n_xn_y \\ -2n_xn_y && 1-2n_y^2 \end{smallmatrix} \right] $
Reflection (3D)
$p_n = S(n, -1) = \left[ \begin{smallmatrix} 1+(-1-1)n_x^2 && (-1-1)n_xn_y && (-1-1)n_xn_z \\ (-1-1)n_xn_y && 1+(-1-1)n_y^2 && (-1-1)n_yn_z \\ (-1-1)n_xn_z && (-1-1)n_yn_z && 1+(-1-1)n_z^2 \end{smallmatrix} \right] $
$ = \left[ \begin{smallmatrix} 1-2n_x^2 && -2n_xn_y && -2n_xn_z \\ -2n_xn_y && 1-2n_y^2 && -2n_yn_z \\ -2n_xn_z && -2n_yn_z && 1-2n_z^2 \end{smallmatrix} \right] $
