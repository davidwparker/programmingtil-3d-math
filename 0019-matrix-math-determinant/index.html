Matrix Math
How to compute the determinant of a matrix part 2
2x2 and 3x3
Every square matrix has a special scalar called the determinant of a matrix
Non-square matrices do not have a defined determinant.
2x2:
$ |M| = \left| \begin{smallmatrix} m_{11} && m_{12} \\ m_{21} && m_{22} \end{smallmatrix} \right| = m_{11}m_{22} - m_{12}m_{21} $
Shortcut: take the diagonal numbers ($m_{11}, m_{22}$), multiply them and then subtract the back diagonal numbers ($m_{12}, m_{21}$) multiplied
2x2 Example:
$ \left| \begin{smallmatrix} 3 && 4 \\ -2 && 1 \end{smallmatrix} \right| = (3)(1) - (4)(-2) = 3 + 8 = 11 $
$ \left| \begin{smallmatrix} a && b \\ c && d \end{smallmatrix} \right| = ad - bc $
3x3:
$ |M| = \left| \begin{smallmatrix} m_{11} && m_{12} && m_{13} \\ m_{21} && m_{22} && m_{23} \\ m_{31} && m_{32} && m_{33} \end{smallmatrix} \right| $
$ = m_{11}m_{22}m_{33} + m_{12}m_{23}m_{31} + m_{13}m_{21}m_{32} $
$ - m_{13}m_{22}m_{31} - m_{12}m_{21}m_{33} - m_{11}m_{23}m_{32} $
$ = m_{11}(m_{22}m_{33}-m_{23}m_{32}) $
$ + m_{12}(m_{23}m_{31}-m_{21}m_{33})$
$ + m_{13}(m_{21}m_{32}-m_{22}m_{31}) $
Shortcut:
$ \begin{smallmatrix} m_{11} && m_{12} && m_{13} && m_{11} && m_{12} && m_{13} \\ m_{21} && m_{22} && m_{23} && m_{21} && m_{22} && m_{23} \\ m_{31} && m_{32} && m_{33} && m_{31} && m_{32} && m_{33} \end{smallmatrix} $
3x3 Example:
$ \left| \begin{smallmatrix} 7 && 1 && -3 \\ -2 && 3 && 2 \\ 1 && 5 && 2 \end{smallmatrix} \right| $
$ = (7)(3)(2)+(1)(2)(1)+(-3)(-2)(5) $
$ - (-3)(3)(1)-(1)(-2)(2)-(7)(2)(5) $
$ = 42+4+30 +9 +4 - 70 = 19 $
