Determinant of the matrix is equal to the "triple product" of the three vectors (three rows of a 3x3 matrix as three vectors):
$ \left| \begin{smallmatrix} a_x && a_y && a_z \\ b_x && b_y && b_z \\ c_x && c_y && c_z \end{smallmatrix} \right| = $
$ = a_xb_yc_z+a_yb_zc_x+a_zb_xc_y - a_zb_yc_x-a_yb_xc_z-a_xb_zc_y $
$ = (a_yb_z-a_zb_y)c_x + (a_zb_x-a_xb_z)c_y + (a_xb_y-a_yb_x)c_z $
$ = (a\times b) \cdot c $
$Minor$ of $M$:
The submatrix $M^{ij}$ that is derived by deleting row $i$ and column $j$ from matrix $M$.
Minor Example:
$ \left[ \begin{smallmatrix} 3 && 4 && 2 \\ -2 && 1 && 1 \\ 1 && 3 && 2 \end{smallmatrix} \right] $
Minor $M^{21} = \left[ \begin{smallmatrix} 4 && 2 \\ 3 && 2 \end{smallmatrix} \right] $
Cofactor of a square matrix:
The signed determinant of the corresponding minor of M
$ c_{ij} = (-1)^{i+j}|M^{ \{ ij \} }| $
Note that a minor is a matrix quantity
Note that a cofactor is a scalar quantity
The term $(-1)^{i+j}$ in the the computation of the cofactor negates every cofactor in a checkboard pattern:
$ \left[ \begin{smallmatrix} + && - && + \\ - && + && - \\ + && - && + \end{smallmatrix} \right] $
We will now compute determinant of a matrix of arbitrary dimension $n$ in terms of its cofactors.
  • Select a row or column for a matrix
  • For each element in the row or column, multiply that element by its cofactor
  • Summing these products yields the determinant
  • Note it's recursive
$ |M| = \sum_{j=1}^n m_{ij}c_{ij} = \sum_{j=1}^n m_{ij}(-1)^{i+j}|M^{ \{ ij \} }| $

$ + m_{13} \left[ \begin{smallmatrix} m_{21} && m_{22} && m_{24} \\ m_{31} && m_{32} && m_{34} \\ m_{41} && m_{42} && m_{44} \end{smallmatrix} \right] - m_{14} \left[ \begin{smallmatrix} m_{21} && m_{22} && m_{23} \\ m_{31} && m_{32} && m_{33} \\ m_{41} && m_{42} && m_{43} \end{smallmatrix} \right] $
Characteristics of Determinants
  • Determinant of matrix product is equal to the product of the determinants:
    $|AB| = |A||B|$ ...even more than one:
    $|M_1M_2...M_{n-1}M_n| = |M_1||M_2|...|M_{n-1}||M_n|$
  • Determinant of the transpose of a matrix is equal to the original determinant
    $|M^T| = |M|$
Characteristics of Determinants
  • If any row or column in a matrix contains all zeros, then the determinant of the matrix is zero
  • Exchanging any pair of rows (or pair of columns) negates the determinant
  • Adding any nonzero multiple of a row/column to another row/column does not change the value of the determinant
