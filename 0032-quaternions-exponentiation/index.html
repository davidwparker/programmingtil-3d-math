Quaternion
Exponentiation
"Raised to a power": $q^t$
(not to be confused with exponential from last episode)
Recall:
$a^0 = 1$ & $a^1 = a$ where $a$ is a nonzero scalar
$0 ... t ... 1 \rightarrow 1 ... a^t ... a$
$0 ... t ... 1 \rightarrow [1,0] ... q^t ... q$
Gives us a fraction of an angular displacement
Beyond $0 ... 1$ range, also normal (**mostly):
$q^2$ is twice the angular displacement of $q$.
**NOTE: quaternions represent angular displacement using the shortest arc. There are no multiple spins. If $q = 30^\circ$ clockwise, then $q^4 \neq 240^\circ$ clockwise.
Instead, it is $80^\circ$ counterclockwise. Rotating $240^\circ$ CW is the same as rotating $80^\circ$ CCW.
Due to this fact, some things aren't trivial:
$(q^4)^{1/2} \neq q^2$

Formal definition
$q^t = exp(t\ log\ q)$
