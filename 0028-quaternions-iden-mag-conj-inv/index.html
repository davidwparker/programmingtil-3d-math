Identity Quaternions
Geometrically: Two identity quaternions, which represent no angular displacement:
$[1, (0,0,0)]$ and $[-1, (0,0,0)]$
Identity Quaternions
When $\theta$ is an even multiple of $360^{\circ}$ then $cos(\theta/2) = 1$ so we have the first form.
When $\theta$ is an odd multiple of $360^{\circ}$ then $cos(\theta/2) = -1$ so we have the second form.
During both $sin(\theta/2) = 0$ so the value of $n$ is unimportant.
Identity Quaternions
Mathematically: one identity quaternion: $[1, (0,0,0)]$. When we have $-q$ (after multiplying by $[-1, (0,0,0)]$, we don't have $q = -q$ are equal, thus, mathematically, $[-1, (0,0,0)]$ is not a mathematical identity
Quaternion Magnitude
Recall the formula for vector magnitude:
$\left\Vert v\right\Vert = \sqrt{\sum_{i=1}^n {v_i}^2}$
The formula is similar for quaternions:
$\left\Vert q\right\Vert = \left\Vert \left[\begin{smallmatrix} w && (x && y && z) \end{smallmatrix}\right] \right\Vert = \sqrt{w^2+x^2+y^2+z^2}$
$ = \left\Vert \left[\begin{smallmatrix} w && v \end{smallmatrix}\right] \right\Vert = \sqrt{w^2 + \left\Vert v \right\Vert^2}$
Quaternion Magnitude
Geometrically, substituting $\theta$ and $n$:
$\left\Vert q\right\Vert = \sqrt{w^2 + \left\Vert v \right\Vert^2} $
$= \sqrt{cos^2(\theta/2) + (sin(\theta/2)\left\Vert n\right\Vert)^2}$
$= \sqrt{cos^2(\theta/2) + sin^2(\theta/2)\left\Vert n\right\Vert^2}$
Quaternion Magnitude
$n$ is unit length:
$= \sqrt{cos^2(\theta/2) + sin^2(\theta/2)(1)}$
$= \sqrt{cos^2(\theta/2) + sin^2(\theta/2)}$
Trig Identity ($cos^2x + sin^2x = 1$)
$= \sqrt{1} = 1$
Note: we'll only be using unit quaternions
Quaternion Conjugate
Conjugate Notation: $q^*$
Obtained by negating the vector portion of the quaternion:
$q^* = \left[\begin{smallmatrix} w && v \end{smallmatrix}\right]^* = \left[\begin{smallmatrix} w && -v \end{smallmatrix}\right] $
$ \left[\begin{smallmatrix} w && (x && y && z) \end{smallmatrix}\right]^* = \left[\begin{smallmatrix} w && (-x && -y && -z) \end{smallmatrix}\right] $
Quaternion Inverse
Inverse notation: $q^{-1}$
$q^{-1} = \frac{q^*}{\left\Vert q\right\Vert}$
Quaternion Inverse Notes
When we multiply a quaternion and its inverse, we get the identity quaternion:
$qq^{-1} = \left[\begin{smallmatrix} 1 && 0 \end{smallmatrix}\right] $
For our purposes, using unit quaternions, conjugate and inverse will be equal.
Conjugate (Inverse) notes:
$q$ and $q^*$ represent opposite angular displacements. When we negate $v$, we negate the axis of rotation $n$. This in turn reverses the "positive rotation". So $q$ rotates about an axis by amount $\theta$ and $q^*$ rotates in the opposite direction by the same amount.
