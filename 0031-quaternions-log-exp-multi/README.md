## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/

## CONCEPTS:

* Quaternion Log
* Quaternion Exp
* Quaternion Multiplaction by scalar
