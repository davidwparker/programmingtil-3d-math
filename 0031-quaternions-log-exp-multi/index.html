Quaternion Log, Exp and
Multiplication by a scalar
Logarithm of $q$
Define $\alpha = \theta/2$
$\left\Vert n \right\Vert = 1$
$q = \left[\begin{smallmatrix} cos\ \alpha && n\ sin \alpha \end{smallmatrix}\right]$
$ = \left[\begin{smallmatrix} cos\ \alpha && (x\ sin\ \alpha && y\ sin\ \alpha && z\ sin\ \alpha) \end{smallmatrix}\right] $
$log\ q = log(\left[\begin{smallmatrix} cos\ \alpha && n\ sin\ \alpha \end{smallmatrix}\right])$
$\equiv \left[\begin{smallmatrix} 0 && \alpha n \end{smallmatrix}\right] $
($\equiv$ means equal by definition). The result of log q is not necessarily a unit quaternion.
Exponential
$p = \left[\begin{smallmatrix} 0 && \alpha n \end{smallmatrix}\right]$
$= \left[\begin{smallmatrix} 0 && (\alpha x && \alpha y && \alpha z) \end{smallmatrix}\right]$
$\left\Vert n \right\Vert = 1$
Exponential
$exp\ p = exp(\left[\begin{smallmatrix} 0 && \alpha n \end{smallmatrix}\right])$
$= \left[\begin{smallmatrix} cos\ \alpha && n\ sin\ \alpha \end{smallmatrix}\right]$
Note that $exp\ p$ always returns a unit quaternion
Note $e^{ln\ a} = a$, $exp(log\ q) = q$.
Multiplication by a scalar
Given scalar $k$
$kq = k[w\ v] = [kw\ kv]$
$= k[w\ (x\ y\ z)] = [kw\ (kx\ ky\ kz)]$
Note: normally doesn't result in a unit quaternion.
