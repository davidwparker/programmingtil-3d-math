Orthographic Projection
Projection refers to a dimension-reducing function.
Projection onto a cardinal axis or plane
Discard one of the dimensions while assigning data into a variable of lesser dimensions.
3D -> 2D by discarding z-values and keeping x and y only.
Projection onto a cardinal axis or plane
Also, just "scale" by zero to remove the extra dimension.
3D matrix projected onto xy plane
$p_{xy} = S([0\enspace0\enspace1], 0) = \left[ \begin{smallmatrix} 1 && 0 && 0 \\ 0 && 1 && 0 \\ 0 && 0 && 0 \end{smallmatrix} \right]$
Projection onto a arbitrary line (2D)
$p_n = S(n, 0) = \left[ \begin{smallmatrix} 1+(0-1)n_x^2 && (0-1)n_xn_y \\ (0-1)n_xn_y && 1+(0-1)n_y^2 \end{smallmatrix} \right] $
$= \left[ \begin{smallmatrix} 1-n_x^2 && -n_xn_y \\ -n_xn_y && 1-n_y^2 \end{smallmatrix} \right] $
Projection onto a arbitrary plane (3D)
$p_n = S(n, 0) = \left[ \begin{smallmatrix} 1+(0-1)n_x^2 && (0-1)n_xn_y && (0-1)n_xn_z \\ (0-1)n_xn_y && 1+(0-1)n_y^2 && (0-1)n_yn_z \\ (0-1)n_xn_z && (0-1)n_yn_z && 1+(0-1)n_z^2 \end{smallmatrix} \right] $
$ = \left[ \begin{smallmatrix} 1-n_x^2 && -n_xn_y && -n_xn_z \\ -n_xn_y && 1-n_y^2 && -n_yn_z \\ -n_xn_z && -n_yn_z && 1-n_z^2 \end{smallmatrix} \right] $
