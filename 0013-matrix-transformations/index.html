Matrix Linear Transformations
Scaling
We can make an object bigger or smaller by scaling by a factor $k$.
If all directions use the same scale, it is called a uniform scale.
If $|k| < 1$, the object gets smaller or shorter in that direction.
If $|k| > 1$, the object gets bigger or longer in that direction.
Next episode:
If $k = 0$ then we have an orthographic projection.
If $k < 0$, then we have a reflection.
Scaling along Cardinal Axes
Two scalars: $k_x, k_y$. Basis vectors $p$ and $q$ are each affected by their respective scalar
$p^\prime = k_xp = k_x[1\enspace0] = [k_x\enspace0]$
$q^\prime = k_yp = k_y[0\enspace1] = [0\enspace k_y]$
$S(k_x,k_y) = \left[ \begin{smallmatrix} k_x && 0 \\ 0 && k_y \end{smallmatrix} \right]$
Scaling along Cardinal Axes
In 3D, add $k_z$
$S(k_x,k_y,k_z) = \left[ \begin{smallmatrix} k_x && 0 && 0 \\ 0 && k_y && 0 \\ 0 && 0 && k_z \end{smallmatrix} \right]$
Scale in an arbitrary direction
$n$ = unit vector parallel to scalar factor $k$
Compute $v^\prime$ in terms of $v, n, k$.
Scale in an arbitrary direction
$v = v_\Vert + v_\perp$
$v_\Vert = (v\cdot n)n$
$v_\perp$ is perpendicular to $n$ = it will not be affected by $k$
$v^\prime = v^\prime_\Vert+v_\perp$
Need $v^\prime_\Vert$, which is just $kv_\Vert$
Scale in an arbitrary direction
$v^\prime = v^\prime_\perp + v^\prime_\Vert$
$v^\prime = v - (v\cdot n)n + k(v\cdot n)n$
$v^\prime = v + (k-1)(v\cdot n)n$
Scale in an arbitrary direction- basis vector 2D
$p = [1\enspace0]$
$p^\prime = p + (k-1)(p\cdot n)n$
$p^\prime = \left[ \begin{smallmatrix} 1 \\ 0 \end{smallmatrix} \right] + (k-1)( \left[ \begin{smallmatrix} 1 \\ 0 \end{smallmatrix} \right] \cdot \left[ \begin{smallmatrix} n_x \\ n_y \end{smallmatrix} \right] ) \left[ \begin{smallmatrix} n_x \\ n_y \end{smallmatrix} \right] $ $p^\prime = \left[ \begin{smallmatrix} 1+(k-1)n_x^2 \\ (k-1)n_xn_y \end{smallmatrix} \right] $
Scale in an arbitrary direction- basis vector 2D
$q = [0\enspace1]$
$q^\prime = \left[ \begin{smallmatrix} (k-1)n_xn_y \\ 1+ (k-1)n_y^2 \end{smallmatrix} \right] $
Scale in an arbitrary direction- basis vector 2D
$S(n,k) = \left[ \begin{smallmatrix} p^\prime \\ q^\prime \end{smallmatrix} \right]$
$ = \left[ \begin{smallmatrix} 1+(k-1)n_x^2 && (k-1)n_xn_y \\ (k-1)n_xn_y && 1+ (k-1)n_y^2 \end{smallmatrix} \right] $
Scale in an arbitrary direction- basis vector 3D
$p = [1\enspace0\enspace0]$
$p^\prime = \left[ \begin{smallmatrix} 1+(k-1)n_x^2 \\ (k-1)n_xn_y \\ (k-1)n_xn_z \end{smallmatrix} \right] $
Scale in an arbitrary direction- basis vector 3D
$q = [0\enspace1\enspace0]$
$q^\prime = \left[ \begin{smallmatrix} (k-1)n_xn_y \\ 1+(k-1)n_y^2 \\ (k-1)n_yn_z \end{smallmatrix} \right] $
Scale in an arbitrary direction- basis vector 3D
$r = [0\enspace0\enspace1]$
$r^\prime = \left[ \begin{smallmatrix} (k-1)n_xn_z \\ (k-1)n_yn_z \\ 1+(k-1)n_z^2 \end{smallmatrix} \right] $
Scale in an arbitrary direction- basis vector 3D
$S(n,k) = $
$ \left[ \begin{smallmatrix} 1+(k-1)n_x^2 && (k-1)n_xn_y && (k-1)n_xn_z \\ (k-1)n_xn_y && 1+(k-1)n_y^2 && (k-1)n_yn_z \\ (k-1)n_xn_z && (k-1)n_yn_z && 1+(k-1)n_z^2 \end{smallmatrix} \right] $
