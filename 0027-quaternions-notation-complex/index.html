Axis-Angle Pair, Negation
Quaternion Notation
  • use four numbers to express orientation
  • contains a scalar component ($w$) + a 3D vector component ($v$), or ($x,y,z$)
  • note: no difference between row vs column notation
$\left[\begin{smallmatrix} w && v \end{smallmatrix}\right]$
$\left[\begin{smallmatrix} w && (x && y && z) \end{smallmatrix}\right]$
Quaternions as Complex Numbers
Quaternions extend the complex number system with three imaginary numbers:
$i^2 = j^2 = k^2 = -1$
$ij = k, ji = -k$
$jk = i, kj = -i$
$ki = j, ik = -j$
$\left[\begin{smallmatrix} w && (x\enspace y\enspace z) \end{smallmatrix}\right] = w+xi + yj + zk $
Quaternions as axis-angle pair
The axis-angle pair $(\theta, n)$ defines the angular displacement of $\theta$ radians about the axis specified by $n$.
The four values of a quaternion:
$q = \left[\begin{smallmatrix} cos(\theta/2) && sin(\theta/2)n \end{smallmatrix}\right] $
$ = \left[\begin{smallmatrix} cos(\theta/2) && (sin(\theta/2)n_x && cos(\theta/2)n_y && sin(\theta/2)n_z ) \end{smallmatrix}\right] $
Quaternion Negation
Quaternions negation is done easily:
The four values of a quaternion:
$-q = -\left[\begin{smallmatrix} w && (x && y && z) \end{smallmatrix}\right] $ $= \left[\begin{smallmatrix} -w && (-x && -y && -z) \end{smallmatrix}\right]$
$=-\left[\begin{smallmatrix} w && v \end{smallmatrix}\right]$ $=\left[\begin{smallmatrix} -w && -v \end{smallmatrix}\right]$
Quaternion Negation
$q$ and $-q$ represent the same angular displacement.
Adding $360^{\circ}$ to $\theta$ doesn't change the angular displacement, but negates all the components of $q$.
