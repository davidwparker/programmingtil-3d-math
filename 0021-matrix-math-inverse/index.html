Matrix Math
Inverse of a matrix
The inverse of a matrix $M$, is the matrix $M^{-1}$, that when we multiply $M$ and $M^{-1}$ on either side, the result is the identity matrix.
$M(M^{-1}) = M^{-1}M = I$
Notes:
Not all matrices have an inverse: if a matrix has a row or column with all zeros, then you'll always end up with a matrix full of zeros when multiplied.
If a matrix has an inverse, it is called invertible or non-singular.
If a matrix has no inverse, it is called non-invertible or singular.
Determinant of a singular matrix is zero.
Determinant of a non-singular matrix is nonzero.
Any invertible matrix $M$, the vector equality $vM = 0$ is only true if $v = 0$

$ -\left| \begin{smallmatrix} 2 && 1 \\ 3 && 1 \end{smallmatrix} \right| = 1 \enspace +\left| \begin{smallmatrix} 4 && 1 \\ -2 && 1 \end{smallmatrix} \right| = 6 \enspace -\left| \begin{smallmatrix} 4 && 2 \\ -2 && 3 \end{smallmatrix} \right| = -16 $

Adjoint of M Example.
$ \left[ \begin{smallmatrix} 6 && 0 && 12 \\ 1 && 6 && -16 \\ -5 && 6 && 8 \end{smallmatrix} \right]^T = \left[ \begin{smallmatrix} 6 && 1 && -5 \\ 0 && 6 && 6 \\ 12 && -16 && 8 \end{smallmatrix} \right] $
Now that we have the classical adjoint, compute the inverse by dividing by the determinant.
$$M^{-1} = \frac{adj M}{|M|}$$

Properties of a matrix inverse:
  • $(M^{-1})^{-1} = M$
  • $I^{-1} = I$
  • $(M^T)^{-1} = (M^{-1})^T$
  • $(AB)^{-1} = B^{-1}A^{-1}$
